# 安装

### 快速开始

```bash
# npm
npm create @whouu/app <project-name>

# yarn
yarn create @whouu/app <project-name>

# pnpm
pnpm create @whouu/app <project-name>

# cli
npm i -g @whouu/create-app

create-app <project-name>
```
