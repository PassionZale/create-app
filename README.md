# create-app

Create an app powered by @whouu/create-app

```bash
npm init @whouu/app <project-name>

cd <project-name>

npm install

npm run dev
```

If using NPX:

```bash
npx @whouu/create-app <project-name>

cd <project-name>

npm install

npm run dev
```

If using Yarn:

```bash
yarn create @whouu/app <project-name>

cd <project-name>

yarn

yarn dev
```
